How to Update Your Website

Quick Update Steps:

  1. Make your changes in the Blogs or Categories tab
  2. Copy the generated code using the "Copy Code" button
  3. Open index.html in a text editor (Notepad++, VS Code, or any code editor)
  4. Find the section to update:
    • For blogs: Search for <!-- Blog Cards -->
    • For categories: Search for <!-- Category Pills -->
  5. Replace the content between the comment markers with your copied code
  6. Save the file
  7. Commit and push to GitHub for automatic deployment

Alternative: Use Git Commands

  1. After updating index.html, open terminal/command prompt
  2. Navigate to your project folder
  3. Run: git add index.html
  4. Run: git commit -m "Update blogs/categories"
  5. Run: git push origin master
  6. Netlify will automatically deploy your changes!

Find the Right Section in index.html

For Blog Updates:

Look for this section: <!-- Blog Cards --> <div class="blog-track"> <div class="blog-cards"> [PASTE YOUR BLOG CODE HERE] </div> </div>

For Category Updates:

Look for this section: <!-- Category Pills --> <div class="blog-categories"> [PASTE YOUR CATEGORY CODE HERE] </div>
